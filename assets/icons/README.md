# Icons

Add two PNG app icons here for the PWA install / home-screen:

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512

They're referenced by `../../manifest.webmanifest`. Until you add them the site
still works; only the install prompt / home-screen icon are affected. Use a
maskable-safe design (keep important content in the centre ~80%).
